Vue-cli配置 - 开启Gzip

1.在config -> index.js中把 build.productionGzip 设置成 true；

2.安装插件 npm i compression-webpack-plugin@1.1.12 -D；

安装最新的话需要 webpack>4, node>8；

3.执行 npm run build 即可；
